CpuHddMem

Назначение

Индикация загрузки процессора, жёсткого диска и памяти на иконке в трее.





Запустить. В трее появится иконка с 3-мя индикаторами: слева процессор, посредине жёсткий диск, справа память ОЗУ


Параметры ini-файла

background=2 ; цвет фона
color_cpu=09F ; цвет индикатора процессора
color_mem=F00 ; цвет индикатора памяти
color_hdd=F90 ; цвет индикатора жёсткого диска
timer=2000 ; интервал между запросами данных в миллисекундах, минимальное 500, ниже не принимается.

Веб цвета вычисляются следующим правилом 09F = 0099FF, 2 = 222222, 3F = 3F3F3F.

Подсказка

Кликните на иконке, чтобы появилась обновляемая подсказка. Закрыть её можно вторым кликом или на кнопке "Закрыть" в подсказке.

Вычисления

Показания загруженности процессора и жёсткого диска вычисляются в среднем. Пиковые значения могут значительно превышать, используется порог, выше которого значения определяются как 100%.

Практика

Полезно программистам следить за состоянием нагрузки на процессор, чтобы отследить неправильный цикл без дела нагружающий процессор.
Может быть полезным для пользователя следить за потребляемой памятью при запуске монструозных приложений, например браузера Firefox + Gimp + ещё что нибудь.
При выключении в спящий режим закрывая программы (Firefox и т.д.) следить, чтобы датчики процессора и жёсткого диска упали близко к 0, дабы не кешировать эти процессы в файл гибернации будучи не успевая выгрузится.
