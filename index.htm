Программы по рейтингу

Все программы на AutoIt3 только под Windows. На Linux они могут быть запущены через Wine.

Программы на PureBasic разделены на 3 раздела:
  • кросплатформенные для Windows и Linux
  • только для Windows
  • только для Linux


Программы для Linux могут быть скомпилированы на "Mac OS", при этом в исходнике нужно поменять все #PB_OS_Linux на #PB_OS_MacOS.

Программы на AutoIt3 были написаны во времена WInXP (2010-2014), поэтому все они тестированы на этой ОС. В большинстве случаев программы работают и на WIn7-10, в описании есть уточнения по этому поводу.
